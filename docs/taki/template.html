linecache.rb

Rubyにおいてキャッシュ を使って、ランダムアクセスをサポートするクラスです。
SingletonをMixinしています。

linecache.tar.bz2(7.24kb) 

使い方例
----------------------------------------
pass = Dir.pwd + "/foo.txt"

line = Linecache.instance
puts line.getline_cache(pass, 3)
puts line.getline_cache(pass, 4)
----------------------------------------
このようにして取得できます。


クリアするときは以下のようにしてください。
----------------------------------------
pass = Dir.pwd + "/foo.txt"

line = Linecache.instance
line.clear
----------------------------------------
