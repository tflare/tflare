Java

Sun Microsystems社によって1995年に発表された。オブジェクト指向を導入している言語
仮想マシン上で動作し、OS・プラットフォームに依存しないことを生かし iアプリなどの携帯電話にも使われている今人気がある言語

システム開発上の利点として、ハードウエアの選定を遅らせることができるというものがある。長期間でのプロジェクト等の場合、開発時と稼動時において、プラットフォームの変更が可能なので、ハードウエア選定を遅らせることができる。これにより開発期間間のハードウエアの陳腐化をなくすことができる。

開発年度:1994

COBOL

事務処理がメインの言語
10進数の型(DECIMAL)があるため、浮動小数点の計算の際でも正確に計算が容易なため、お金を扱う必要性があるシステムで利用される。
COBOLは習得がたやすい言語のため、新人が送り込まれたり、他の言語を習得できなかった人が送り込まれてプロジェクトがデスマーチに陥りやすい。
またCOBOLは保守性の悪いプログラムを書きやすい言語なので保守が大変。
開発年度:1959

Lisp

John McCarthyらによって開発された関数型のプログラミング言語
問題解決にあったミニ言語を作り、ミニ言語に基づいてプログラムを作成することが可能。
独自の拡張の結果、数多くの方言が出来てしまい互換がとれなくなってしまった。統一化が図られたのがCommon-LISPで、教育用にシンプルに再構築されたのがSchemeである。
開発年度:1958
