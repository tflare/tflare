オペレーションシステム(OS)

パソコンの基本的な操作法や使えるソフトがこれによって決定される。
基本ソフトともいわれる。
OSはハードウェアの各種資源の管理とハードウェアが効率的に使用できるような、制御プログラムを提供する。
また、パソコンのアプリケーションソフトで共通する動作をAPIにより提供し、それらの操作を共通化せさせる。

Windows

個人向け

Windows 95 → Windows 98 → Windows 98SE → Windows ME → Windows XP personal → Windows Vista → Windows 7

企業・サーバー向け

Windows NT 3.1 → Windows NT 3.5X → Windows NT 4.0 → Windows 2000 → Windows XP → Windows Server 2003 → Windows Server 2008 → Windows Server 2008 R2

Windows XPによってマイクロソフトにとって悲願だったWindows 95系とWindows NT系との融合を果たしたことになる。
これによりマイクロソフトは開発費の軽減などができるとしている。

MacOS

オペレーションシステムの一つ。
最新のMacOSX （マックオーエステン）が発売。
MacOSX はMacOSが作られて以来始めてコードが一から書き直された。
MacOSX はAquaというユーザーインターフェイスを持っている。
Aquaの特徴の一つとして半透明のメニューやアイコン、影付きのウインドウがあげられる。
MacOSX はコードが一から書き直されたため、
今までのソフトがそのままでは動かないので、
Classicという今までのソフトが動く環境を用意している。
OSやパソコン本体のデザインの良さも人気の一つとしてあげられる。
グラフィックや音楽を昔からやっている人に人気が高い。
バージョン10.1によってDVDの再生がサポートされた。
バージョン10.2でQuartzにおける最適化、付属emailソフトのスパム対応強化、GCC 3.1搭載、Rendezvous搭載 などかなりの機能強化がなされた。Rendezvousとは、コ ンピュータ、プリンタ、その他の周辺機器のネットワークにつなげることが可能なもの

Linux

主にサーバーといわれる環境で使われる。
安定性が高い。ソースが公開されている。という特徴がある。

ディストリビュ−ションパッケージとは、配布パッケージという意味でOSとして使えるようにまとめたもの。

Android

スマートフォンやタブレットPCで使用されているオペレーションシステム
携帯デバイス向けのOS
カーネルにLinuxの関連技術が使用されている
開発環境としてLinux、Mac OS X、Windowsをサポートする。
アプリケーション配布用アプリケーションでは25ドルの登録手数料が初回のみ必要なだけであり開発に対する敷居が低い
