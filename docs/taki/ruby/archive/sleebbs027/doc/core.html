SleBbsの実装の細かい話

レス番号

SleBbsではレスファイルにレス番号を保存していない
入れなくてもできるんじゃないのと思ったからだ。
確かにできたのだがResreadクラスとResHtmlクラスが複雑になってしまった。
レス番号を表示する必要がある場合はレス番号を保存しといたほうが楽ができる。

SleBbs モジュール

明らかにSleBbsモジュールに入れたほうがよいものでも今は入っていない。
libの中にもSleBbsモジュールに入れたほうがよいものとplugin扱いにしたほうがよいものに分かれると思う。

ログの記録

現在は1スレッド1ファイルにしてしているのだが、
ツリー式掲示板としても扱えるようにしようと考えていたら、今の仕様だと厳しい事がわかった。
スレッド式の場合スレッド数は抑えられるのが一般的だけど
ツリー式の場合、ツリーのレス数が少なくツリーの数が多くなるのが一般的だ。
1ツリーごとに1ファイル作っていたら管理が大変になると思われるので、
次のバージョンでは今の仕組みを変えるつもりだ。
