機能概説

ページごとの操作

編集

編集のリンクをクリックすることにより、内容を編集することができます。
内容が空のままSubmitボタンを押すとページが削除されます。
ページを選択していなければ編集のリンクの有効化されません。

履歴

履歴のリンクをクリックすることにより、履歴を表示することができます。
表示される世代数は変更することができます。
設定を変えていなければは3世代分表示されます。

ページに関連しない操作

新規

新規のリンクをクリックすることにより、新たにページを作成します。

トップ

トップのリンクをクリックすることにより、トップページを表示します。
トップページは管理ページで変更することができます。

一覧

一覧のリンクをクリックすることにより、ページの一覧を表示します。

更新順

更新順のリンクをクリックすることにより、ページの一覧を更新順で表示します。

管理

ユーザ名とパスワードの変更が行えます。
初期ユーザ名:test、初期パスワード:testです。

Search

検索を行います。文字列一致検索を行います。
該当部分とページへのリンクが表示されます。

機能説明

url自動リンク

urlらしい文字列は自動的にリンクされます。

オートリンク

他のページ名があると自動でリンクされる機能。
無効にすることもできます。

認証

編集と新規に認証をかけることができます。
無効にすることもできます。
