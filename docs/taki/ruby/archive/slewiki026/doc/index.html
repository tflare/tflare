SleWikiとは？

SleWikiには以下のような機能があります。
http://,https://,ftp:// から始まる文字列に自動でリンクが張られます。
登録した単語があれば、該当の単語のページにリンクされます。この機能は無効にもできます。
認証機能があります。

ポリシーとして、覚えずらい機能は搭載しない。というのがあります。
独自の仕様を搭載しても覚えられないし、覚えたとしてもすぐに忘れてしまいます。
複雑なことをしなくても、良きように計らってほしいです。

[動作環境]

Ruby1.6.8, Ruby1.8.2で動作確認しています。
